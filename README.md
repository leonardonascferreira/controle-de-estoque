## Sistema de Controle de Estoque 🛒

Este é um **sistema de controle de estoque** desenvolvido em **Python**, com interface no terminal e **persistência de dados utilizando JSON**.  
O projeto foi desenvolvido como **projeto acadêmico**, com foco na aplicação prática de lógica de programação e melhoria da experiência do usuário em aplicações de linha de comando.

---

## Funcionalidades 🛠️

* Adicionar produtos com nome, preço e quantidade;
* Atualizar preços ou quantidades de produtos existentes;
* Excluir produtos do estoque com confirmação do usuário;
* Visualizar todo o estoque de forma clara e organizada no terminal;
* Salvar os dados automaticamente em um arquivo JSON, garantindo que o estoque seja mantido mesmo após fechar o programa.

---

## Tecnologias e Conceitos 💻

* **Python** – Lógica de programação, estruturas condicionais e de repetição;
* **Listas e Dicionários** – Organização e manipulação dos dados;
* **JSON** – Leitura e escrita de dados persistentes;
* **Tratamento de erros** – Uso de `try` e `except` para evitar falhas ao inserir dados inválidos;
* **UX em terminal** – Uso de cores e mensagens para melhorar a interação com o usuário.

---

## Como usar ▶️

1. Clone este repositório:

```bash
git clone <URL_DO_REPOSITORIO>
```

2. Execute o arquivo Python no terminal:

```bash
python sistema_estoque.py
```

3. Siga as instruções do menu para gerenciar o estoque.

---

## Estrutura do arquivo JSON

O arquivo estoque.json possui a seguinte estrutura:

```json
[
  {
    "nome": "Produto Z",
    "preco": 129.99,
    "quantidade": 20
  },
  {
    "nome": "Produto X",
    "preco": 50.00,
    "quantidade": 10
  }
]
```
